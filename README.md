# Flask Leads App 🚀

Простое веб-приложение на Flask для сбора заявок.

## 🔧 Функционал

- Форма записи (имя + телефон)
- Валидация данных
- Сохранение заявок в CSV
- Страница "Спасибо"
- Админка со списком заявок

## 🖥️ Технологии

- Python
- Flask
- HTML (Jinja2)
- CSV

## ▶️ Как запустить

```bash
python app.py
