# Lead CRM

Небольшое Flask-приложение для сбора заявок.

## Возможности

- Добавление заявок
- Просмотр в админке
- Редактирование
- Удаление
- Поиск
- Авторизация

## Технологии

- Python
- Flask
- SQLite
- HTML/CSS
- Git

## Установка

```bash
git clone <repo_url>
cd project
pip install -r requirements.txt
python app.py
